# GAZON — GameProject

Симулятор работы в ПВЗ (пункт выдачи заказов), чёрная комедия, от первого лица.
Портируется из HTML/Three.js MVP (`../GAZON PVZ Simulator.dc.html`) в Unity 6, целевая платформа — PC/Steam.

## Что уже есть в этой папке
- `Docs/GDD.md` — механики, извлечённые из MVP построчным разбором кода.
- `Docs/Architecture.md` — план AI-first экосистемы разработки (MCP-серверы, веб-панель) по фазам,
  с обоснованием порядка.
- `Docs/Decisions.md` — журнал решений.
- `Design/` — черновики по миру, экономике, клиентам (дополняет GDD, не дублирует).
- `AI/` — правила и контекст для Claude Code (или любого другого агента) в этом репозитории.
- `Database/schema.sql` + `Database/project.sqlite` — база игрового контента (архетипы клиентов,
  реплики, штрафы, мемы, апгрейды донат-магазина, товары бабушки, звания), пересобирается через
  `python3 Tools/ai_suite/init_db.py`.
- `.claude/CLAUDE.md` — инструкции для Claude Code.
