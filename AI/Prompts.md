# Повторяющиеся промпты

Место для промптов/команд, которые оба разработчика используют регулярно и не хотят каждый раз
формулировать заново. Пусто на старте — заполняется по мере того, как что-то реально повторяется
(не сочиняем впрок).

Пример формата записи:

```
## Название задачи
Когда использовать: ...
Промпт:
"""
...
"""
```
