# Quests

В MVP нет системы квестов — только цикл «день → смена → итог» и линейная финальная цель
(накопить 400 000 ₽ на «точку ПВЗ ирл»). Папка оставлена по общей структуре архитектуры
(`Docs/Architecture.md`) на случай, если в Unity-версии появится квестовый слой
(например, побочные договорённости с курьером, ветки лояльности к конкретным клиентам).
Пусто до тех пор, пока это не станет реальным решением, зафиксированным в `Docs/Decisions.md`.
