# Архетипы клиентов (черновик поверх GDD)

Подробности механик — `Docs/GDD.md#Особые-типы-клиентов`. Здесь — заметки для расширения контента,
которые пойдут в `customer_archetypes` при заполнении `Database/project.sqlite`.

| id | Архетип | Шанс спавна | Особенность |
|---|---|---|---|
| `normal` | Обычный | ~71% (1 − babka − shopaholic) | 22% без кода → «Глаз Бога» |
| `babka` | Бабушка | 10% | мини-игра «Крокодил», выше базовое терпение (44 vs 34) |
| `shopaholic` | Шопоголичка-восьмикоробочница | 9% | гарантированная примерка после выдачи |

## Идеи для расширения (не решено, не реализовывать без записи в Decisions.md)

- Архетип «блогер» — снимает сторис на кассе, влияет на рейтинг сильнее обычного.
- Архетип «дед с претензией к каждому болту» — аналог бабушки, но требует не угадать товар,
  а выдержать долгий монолог без потери терпения.
- Любое новое имя реплики/архетипа сначала регистрируется через Project MCP (когда появится,
  см. `AI/MCP.md`), а до тех пор — просто через прямой INSERT в `project.sqlite` с проверкой на дубли.
